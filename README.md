# ATM AI Dashboard (secondAIProjectThales)

Ce projet comporte une application **Next.js** (Frontend) et une API **FastAPI** (Backend).

## Démarrage rapide

Pour installer les dépendances et démarrer les serveurs du Frontend et du Backend simultanément, exécutez simplement le script suivant depuis la racine du projet :

```bash
./start.sh
```

Ce script va :
1. Installer les dépendances npm pour le frontend Next.js.
2. Créer l'environnement virtuel Python et installer les dépendances pour le backend FastAPI.
3. Lancer simultanément :
   - Le **Frontend** sur [http://localhost:3000](http://localhost:3000)
   - Le **Backend** sur [http://localhost:8000](http://localhost:8000)

Appuyez sur `CTRL+C` dans le terminal pour arrêter proprement les deux serveurs.
